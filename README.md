# PR Intelligence & Market Insights Dashboard

## Overview

An end-to-end Power BI dashboard that transforms PR and media intelligence data into actionable business insights. The dashboard helps communication and marketing teams monitor media performance, benchmark channels, analyze sentiment, and identify growth opportunities.

## Business Problem

PR teams generate thousands of media records every month, but most reports remain static and descriptive. This project converts raw media data into interactive dashboards that support strategic decision-making.

## Project Objectives

- Track PR performance across channels
- Monitor Share of Voice
- Analyze sentiment
- Identify top-performing media sources
- Generate strategic recommendations

## Tech Stack

- Power BI
- Excel
- DAX
- Power Query

## Key Features

- KPI Cards
- Share of Voice Analysis
- Channel Performance
- Trend Analysis
- Sentiment Tracking
- Strategic Recommendations

## Business Impact

- Improved visibility into media performance
- Identified high-performing channels
- Highlighted sentiment risks
- Enabled data-driven PR planning
